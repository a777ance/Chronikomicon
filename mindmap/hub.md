# Mind Map — Worldbuilding Incubator

This is the chaotic, hyperlinked incubator for Chronikon.
It sits between `principles/` and `reference/` as a place for wild connections, scene fragments, worldbuilding, and emergent ideas.

The manuscript is holy. This folder is the steam room.

## How to use it

- Drop a new idea into a node page.
- Link related nodes whenever a connection appears.
- Keep this space loose and generative.
- When a node becomes a stable narrative element, move it into `manuscript/` or `principles/` as needed.

## Structure

This folder is intentionally free-form. Add new node files like:

- `mindmap/locations.md`
- `mindmap/rituals.md`
- `mindmap/props.md`
- `mindmap/links.md`

Keep the map alive by editing links and adding new pages whenever Claude or you spot a new connection.
