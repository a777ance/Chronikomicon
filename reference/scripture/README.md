# Scripture Reference

## Sola Scriptura

The text in this directory is the Word of God. It is committed once and never modified. No commentary, annotation, or editorial addition is permitted within the text files.

Artwork files are separate and do not alter the text.

---

## Versions Included

| File | Version | Status |
|------|---------|--------|
| `kjv.txt` | King James Version (1611) | **Committed — do not modify** |

KJV is the primary and sole committed translation. 31,100 verses. 66 books. Public domain.

ASV (1901) and WEB were planned as secondary versions but reliable source files could not be located. KJV is sufficient as the canonical reference.

---

## Text Format

One verse per line:

```
GEN 1:1	In the beginning God created the heaven and the earth.
GEN 1:2	And the earth was without form, and void...
```

Fields: `BOOK CHAPTER:VERSE<tab>text`

Book abbreviations follow the standard 3-letter scheme (GEN, EXO, LEV, ... REV).
