# Soundtrack

Music paired with chapters, scenes, and moods in Chronikon.

This is reference material — not a playlist app, but a map. Each entry connects a piece of music to a specific chapter, moment, or emotional register in the novel. When writing, open the relevant entry and put the song on before you start.

---

## How to use this

1. Before a writing session, check the chapter's soundtrack entry
2. Put the song on
3. Let it run on repeat while you write the scene
4. If a song stops working, note it — the mismatch is information

---

## Adding entries

Anyone (you or Claude) can add to `manifest.md`. Fields:
- **Chapter** — which chapter or scene
- **Song / Artist** — title and artist
- **Mood** — one or two words
- **Cinematic quality** — what it makes you see or feel spatially/visually
- **Notes** — why it belongs here; what it unlocks

Entries are never deleted, only added or annotated.
