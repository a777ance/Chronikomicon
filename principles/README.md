# Principles

This layer is for theory-crafting — organizing frameworks, structural logics, thematic lenses.

Principles are **not** the novel. They are instruments for thinking about the novel. Any principle here can be:

- drawn on freely
- held in tension with another principle
- discarded
- recycled in a new form
- reconfigured mid-process

No principle is binding. The manuscript is the only authority.
